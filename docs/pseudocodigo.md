# Pseudocódigo do projeto

## Objetivo

Descrever a lógica inicial do sistema de gestão de ocorrências técnicas.

## Estrutura prevista

1. Apresentar menu principal.
2. Permitir inserir nova ocorrência.
3. Permitir listar ocorrências registadas.
4. Permitir pesquisar uma ocorrência.
5. Permitir alterar o estado de uma ocorrência.
6. Guardar dados em ficheiro.
7. Carregar dados guardados.
8. Sair do programa.

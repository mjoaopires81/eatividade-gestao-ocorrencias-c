## 5. Instruções de compilação e execução

Para compilar o programa, pode ser usado o seguinte comando:

```bash
gcc main.c -o gestao_ocorrencias
```

Para executar o programa em Linux ou macOS:

```bash
./gestao_ocorrencias
```

Em Windows, depois da compilação, o programa pode ser executado através de:

```bash
gestao_ocorrencias.exe
```

Caso o grupo utilize vários ficheiros `.c`, por exemplo `main.c` e `ocorrencias.c`, deverá adaptar o comando de compilação:

```bash
gcc main.c ocorrencias.c -o gestao_ocorrencias
```
